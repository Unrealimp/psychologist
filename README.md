
  # Personal Website for Psychologist

  This is a code bundle for Personal Website for Psychologist. The original project is available at https://www.figma.com/design/qwqS9pEbHC8nVGxkP2GYCO/Personal-Website-for-Psychologist.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ### Admin password

  Create a `.env` file with `VITE_ADMIN_PASSWORD` (see `.env.example`) to set the admin password.

  ### Production build

  Run `npm run build` to create the production build.

  Run `npm run preview` to serve the production build locally.
